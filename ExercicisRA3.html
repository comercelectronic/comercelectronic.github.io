Sobre aquest curs Teoria Exercicis Aplicació a projectes

CA1. Reconeix la funcionalitat del llenguatge d’etiquetes d’hipertext (HTML).

Exercici 1

Crea un document HTML bàsic amb un títol i un paràgraf. Utilitza etiquetes com <html>, <head>, <body>, <h1>, i <p>.

Objectiu: Familiaritzar-se amb les etiquetes HTML bàsiques.

Exercici 2

Dissenya una pàgina HTML que inclogui una llista ordenada i una llista desordenada, un enllaç a una altra pàgina web, i una taula amb almenys 3 files i 3 columnes.

Objectiu: Aplicar coneixements HTML per crear una pàgina web amb estructura mitjana.

Exercici 3

Desenvolupa una pàgina HTML completa amb capçalera, cos, i peu de pàgina. Inclou enllaços a dues pàgines internes fictícies, afegeix formularis simples amb camps de text i botons, i utilitza etiquetes HTML5 com <header>, <footer>, i <form>.

Objectiu: Crear una pàgina web completa amb una estructura avançada i funcionalitats addicionals.

CA2. Utilitza programes específics per crear els fitxers que componen les pàgines web.

Exercici 1

Utilitza un editor de text simple (com Notepad o TextEdit) per crear un fitxer HTML i un fitxer CSS bàsic. Guarda'ls en una carpeta i crea un document explicant com ho has fet.

Objectiu: Familiaritzar-se amb la creació de fitxers web utilitzant eines bàsiques.

Exercici 2

Utilitza un editor de codi com Visual Studio Code o Sublime Text per dissenyar una pàgina web senzilla. Inclou fitxers HTML i CSS i utilitza el control de versions (per exemple, Git) per gestionar el projecte.

Objectiu: Desenvolupar habilitats amb eines i editors més avançats per crear pàgines web.

Exercici 3

Crea una pàgina web utilitzant un framework com Bootstrap o Tailwind CSS. Inclou components com una barra de navegació, una galeria d'imatges i una secció de testimonis, i configura les opcions de desplegament bàsic.

Objectiu: Utilitzar frameworks específics per dissenyar pàgines web més complexes.

CA3. Registra l'adreça de pàgines web amb domini propi o amb allotjament gratuït.

Exercici 1

Investiga i escriu una breu descripció dels passos per registrar un domini utilitzant un servei de registre de dominis com GoDaddy o Namecheap.

Objectiu: Entendre el procés de registre de dominis.

Exercici 2

Registra un domini amb un servei d'allotjament gratuït com WordPress.com o Wix, i configura una pàgina web de demostració amb el domini registrat.

Objectiu: Aplicar el procés de registre de domini i configurar una pàgina web en un entorn d'allotjament gratuït.

Exercici 3

Registra un domini amb un servei de pagament i configura el teu propi allotjament web. Inclou la configuració de DNS, el desplegament de fitxers web al servidor i la comprovació del funcionament del domini.

Objectiu: Gestionar el registre de dominis i l'allotjament web en un entorn més professional.

CA4. Envia els fitxers web creats al servidor d'Internet mitjançant programes especialitzats en aquesta tasca.

Exercici 1

Utilitza un client FTP bàsic com FileZilla per pujar un fitxer HTML i CSS al teu servidor d'allotjament gratuït. Crea una breu guia explicativa del procés.

Objectiu: Familiaritzar-se amb l'ús d'un client FTP per pujar fitxers.

Exercici 2

Utilitza un client FTP per pujar una sèrie de fitxers d'una pàgina web completa a un servidor d'allotjament de pagament. Inclou la creació d'un document que expliqui el procés de configuració del servidor i la gestió de permisos.

Objectiu: Gestionar el procés de pujada de fitxers i configuració del servidor web.

Exercici 3

Configura i utilitza un entorn de desplegament automatitzat com GitHub Pages o Netlify per desplegar una pàgina web. Inclou la configuració de continuïtat d'integració i desplegament automàtic.

Objectiu: Implementar tècniques avançades de desplegament i automatització de pàgines web.

CA5. Fa servir programes específics d'inclusió de textos, imatges i so.

Exercici 1

Crea una pàgina HTML que inclogui un text, una imatge i un fitxer d'àudio utilitzant etiquetes HTML bàsiques com <img> i <audio>.

Objectiu: Integrar contingut multimèdia bàsic en una pàgina web.

Exercici 2

Dissenya una pàgina web amb text, imatges i un vídeo integrat. Utilitza eines de disseny web per ajustar el format i el posicionament dels elements.

Objectiu: Aplicar tècniques de disseny per incloure i formatar contingut multimèdia.

Exercici 3

Utilitza un editor web com Adobe Dreamweaver per crear una pàgina amb contingut dinàmic, incloent text, imatges i àudio. Implementa eines de gestió de contingut com WordPress per afegir i editar aquest contingut.

Objectiu: Desenvolupar habilitats en l'ús d'eines professionals per gestionar contingut multimèdia.

CA6. Construeix una pàgina web eficient per al comerç electrònic.

Exercici 1

Crea una pàgina de producte simple amb una descripció, preu i imatge del producte. Utilitza etiquetes HTML bàsiques per estructurar la informació.

Objectiu: Entendre les bases d'una pàgina web orientada al comerç electrònic.

Exercici 2

Dissenya una pàgina web amb una secció de productes, un carret de la compra bàsic, i un formulari de contacte. Utilitza CSS per estilitzar la pàgina i assegurar la coherència del disseny.

Objectiu: Crear una pàgina de comerç electrònic amb funcionalitats bàsiques i estilitzat.

Exercici 3

Desenvolupa una pàgina web de comerç electrònic completa amb funcionalitats avançades com passarel·les de pagament, integració amb serveis d'inventari, i optimització per a mòbils. Implementa tècniques d'optimització per a la velocitat i seguretat.

Objectiu: Crear una solució de comerç electrònic completa i optimitzada.

CA7. Inclou en el web enllaços d'interès capaços de generar tràfic orientat i interessat en allò que s’hi ofereix.

Exercici 1

Afegeix enllaços a les teves pàgines web a altres pàgines internes i externes. Utilitza l’etiqueta <a> per crear enllaços i escriu una breu descripció del seu ús.

Objectiu: Entendre com afegir enllaços bàsics en una pàgina web.

Exercici 2

Implementa enllaços a pàgines web relacionades i crea una secció de recursos amb enllaços a articles, blogs o estudis de cas rellevants. Utilitza tècniques de SEO per optimitzar els enllaços.

Objectiu: Crear una xarxa d'enllaços útils i optimitzar-los per al SEO.

Exercici 3

Desenvolupa una estratègia de link building, incloent la creació d’enllaços interns i externs, la col·laboració amb altres llocs web per obtenir enllaços de qualitat, i l’anàlisi del tràfic generat per aquests enllaços.

Objectiu: Implementar una estratègia avançada per augmentar el tràfic web mitjançant enllaços.
