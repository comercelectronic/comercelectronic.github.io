Sobre aquest curs Teoria Exercicis Aplicació a projectes

1.1 Estructura d'una pàgina web corporativa. El llenguatge HTML

  • Estructura bàsica: Capçalera (header), cos (body), peu de pàgina (footer).
  • HTML: Utilització d'etiquetes com <html>, <head>, <body>, etc.
  • Doctype: Definir <!DOCTYPE html> per a versions modernes.

1.2 Reconeixement de pàgines web amb els editors web més usuals
  • WYSIWYG: Editors visuals com Adobe Dreamweaver, WordPress, Wix.
  • Característiques: Personalització, plantilles, compatibilitat amb serveis.

1.3 Creació de pàgines web amb els editors web més usuals
  • Creació de continguts: Textos, imatges, estil CSS, format i color.
  • Hipervincles: Enllaços amb l'etiqueta <a>.
  • Interactivitat: Menús, carrousels, formularis amb <input>.
  • Multimèdia: Vídeos, àudio, imatges animades.

1.4 Elecció del servidor per allotjar pàgines web
  • Allotjament propi: Més control, però més costos i manteniment.
  • Allotjament gratuït: Menys control i personalització, però sense cost.

1.5 Publicació de pàgines web via FTP
  • FTP (File Transfer Protocol): Protocol per transferir fitxers des de l'ordinador al servidor web.
  • Passos: Utilitzar un client FTP com FileZilla, connectar al servidor i transferir els fitxers.

1.6 Alta en cercadors
  • Indexació: Enviar el lloc a cercadors com Google mitjançant Google Search Console.
  • Sitemap: Crear i enviar un fitxer sitemap.xml per ajudar en la indexació.
  • Optimització SEO: Títols, descripcions meta, ús de paraules clau.

1.7 Programes de disseny gràfic i altres utilitats per al web
  • Programes de disseny: Adobe Photoshop, Illustrator, Canva per a la creació de gràfics i imatges.
  • Altres eines: Editors de codi com Visual Studio Code, optimitzadors d'imatges, etc.
  • Manteniment: Actualitzar periòdicament el contingut, millorar seguretat i funcionalitat.

1.8 Catàleg en línia
  • Verificació i processament: Assegurar que els productes i preus són correctes i actualitzats.
  • Efectivitat: Filtres i categories clares per millorar la navegació.
  • Ambient de compra: Crear confiança amb descripcions clares, seguretat en els pagaments.

1.9 Fluxos de caixa de la botiga en línia

  • Ingressos: Provenen de les vendes realitzades en línia.
  • Despeses: Inclouen manteniment, publicitat, enviament i gestió de l'inventari.
  • Passarel·les de pagament: Sistemes com PayPal o Stripe.

1.10 Zones calentes i zones d'usuari
  • Zones calentes: Àrees amb major interacció identificades amb eines d'analítica web.
  • Zones d'usuari: Àrees personalitzades per a usuaris registrats (com historial de comandes).

1.11 El carretó de la compra en línia
  • Funcionalitat: Afegir productes, gestionar la compra i mostrar el preu total.
  • Disseny: Ha de ser intuïtiu i facilitar el procés de compra.
